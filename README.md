# 信创网络安全能力协同分析系统

基于Golang开发的信创网络安全产品能力分析系统，用于分析网络拓扑中安全能力的冗余与空缺。

## 功能特性

- 🔧 **产品管理** - 支持防火墙、IDS、WAF等11类信创安全产品
- 📊 **拓扑分析** - 智能分析网络拓扑中的安全能力冗余与空缺
- 🎯 **场景模板** - 内置企业、政务、金融等10种典型安全场景
- 📈 **可视化** - 提供拓扑可视化数据接口

## 技术栈

- **后端**: Go + Gin + GORM + SQLite
- **前端**: Vue3 + TypeScript（开发中）

## 快速开始

```bash
# 安装依赖
go mod tidy

# 运行项目
go run main.go
```
